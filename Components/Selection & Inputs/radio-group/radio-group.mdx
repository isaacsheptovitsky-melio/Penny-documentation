# Radio Group

import { Tabs, TabItem } from '@astrojs/starlight/components';

[Web URL](https://zeroheight.com/98bb1df01/v/latest/p/79a7b5-radio-group)
The Radio Group component allows users to select a single option from a list. It follows standard radio button behavior but includes an option for customizable content.

```jsx
// Basic Radio Group
<RadioGroup name="payment-method" defaultValue="credit-card">
  <RadioOption value="credit-card">Credit Card</RadioOption>
  <RadioOption value="bank-transfer">Bank Transfer</RadioOption>
  <RadioOption value="paypal">PayPal</RadioOption>
</RadioGroup>

// With Form Field
<FormField label="Select Payment Method" required>
  <RadioGroup name="payment" defaultValue="card">
    <RadioOption value="card">Credit/Debit Card</RadioOption>
    <RadioOption value="ach">ACH Transfer</RadioOption>
    <RadioOption value="check">Check</RadioOption>
  </RadioGroup>
</FormField>

// With Custom Content
<RadioGroup name="subscription" orientation="vertical">
  <RadioOption value="basic">
    <div>
      <h4>Basic Plan</h4>
      <p>$9.99/month - Essential features</p>
    </div>
  </RadioOption>
  <RadioOption value="pro">
    <div>
      <h4>Pro Plan</h4>
      <p>$19.99/month - Advanced features + support</p>
    </div>
  </RadioOption>
</RadioGroup>
```

## Usage

* Use when a user must choose one option from a predefined set.
* Ensure all options are clearly labeled and easy to understand.
* Use the customizable content to provide additional context when needed.
* Group related radio buttons together with a clear heading
