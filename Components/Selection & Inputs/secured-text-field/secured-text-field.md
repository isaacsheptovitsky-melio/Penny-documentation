# Secured text field
[Web URL](https://zeroheight.com/98bb1df01/v/latest/p/502a15-secured-text-field)
The Secured Text Field component is designed to mask sensitive information such as passwords and account numbers. It provides an optional toggle to reveal or hide the entered text.
