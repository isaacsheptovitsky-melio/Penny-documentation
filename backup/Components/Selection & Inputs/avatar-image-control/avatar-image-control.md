# Avatar image control
[Web URL](https://zeroheight.com/98bb1df01/v/latest/p/007711-avatar-image-control)
The Avatar image control displays a user or company, for recognition and personalization. 


| [Storybook](https://penny.melio.com/?path=/story/selection-inputs-components-avatar-image-control--main) |
| --- |

## Usage

* Upload an avatar image in order to represent a user
* Use the Circle variant to represent a user
* Use the Square variant to represent a company
* Upon upload, user is able to delete photo
