# Pagination
[Web URL](https://zeroheight.com/98bb1df01/v/latest/p/69239c-pagination)
Pagination helps users navigate large data sets, such as tables of payments or invoices, by dividing content into manageable pages.

## Usage

* Use pagination at the **bottom of a table or list** when the total number of items exceeds the page size (e.g., 25, 50, or 100)
