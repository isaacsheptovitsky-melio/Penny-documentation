## Usage

* Use the overlay to help focus user attention on a specific task
* Use with the `isLoading` prop to mask a screen / component while data is being fetched, such as verifying a vendor's bank details

## Variants

### Appearance

#### Light

A subtle white obscuring, best for disabling specific items like a form field or table row.

#### Dark

A dark overlay, typically used for full-screen overlays like Modals

#### Darker

A darker overlay color, rarely used.

## Props

### Loading

Displays the Loader component centered within the blanket.

### With Icon

Displays a specific icon centered in the blanket, useful for upload states or empty states.

## Use cases

### Component blanket

The blanket with an icon covers the file attachment, indicating it requires an image file.

## Desktop usage and treatment

The Blanket component can be used both as a full-screen overlay or to cover specific components. When used as a full-screen overlay, it typically employs the dark variant to create proper focus on modal dialogs or other overlays.

## Mobile usage and treatment

On mobile devices, the Blanket component functions identically to desktop, providing the same overlay functionality to obscure background content and focus user attention.
