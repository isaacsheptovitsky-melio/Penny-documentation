## Usage

- Provide clear labels for table columns
- Enable sorting functionality for data columns
- Support filtering and search capabilities
- Display column-specific actions and controls
- Indicate column data types through appropriate styling

## Variants

### Basic header

Simple text labels for columns that don't require interaction.

### Sortable header

Headers with sorting indicators that allow users to order data in ascending or descending order.

### Interactive header

Headers with additional controls like filters, menus, or action buttons.

### Icon headers

Compact headers using icons instead of text, typically for narrow columns or action columns.

## States

### Default

The standard appearance when no interaction is taking place.

### Hover

Visual feedback when users hover over sortable or interactive headers.

### Active

Indicates the current sort column or selected filter state.

### Loading

Shows when column data is being processed or refreshed.

## Desktop usage and treatment

### Text alignment

- Left-align text headers for consistency
- Center-align numeric column headers when appropriate
- Right-align headers for currency or numerical data when needed

### Interactive elements

- Provide clear visual indicators for sortable columns
- Use consistent spacing for header controls
- Ensure adequate touch targets for interactive elements

## Mobile usage and treatment

### Responsive considerations

- Maintain readable header text on smaller screens
- Ensure interactive elements remain accessible
- Consider abbreviated labels or icons for narrow columns
- Provide alternative interaction methods for complex headers
