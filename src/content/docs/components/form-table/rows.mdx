## Usage

- Group related data elements into logical records
- Provide clear visual separation between different data entries
- Support various interaction states for selection and actions
- Enable expandable content for detailed information

## Row types

### Standard rows

Basic rows containing regular table data with consistent spacing and alignment.

### Expandable rows

Rows that can reveal additional content or nested information when activated.

### Group header rows

Special rows used to categorize or group related data entries.

### Summary rows

Rows containing calculated totals, averages, or other aggregated data.

## States

### Default

Standard appearance for normal data display.

### Hover

Visual feedback when users move their cursor over the row.

### Selected

Indicates when a row is selected for bulk actions or detailed operations.

### Expanded

Shows the state when expandable rows are opened to reveal additional content.

### Loading

Displays when row data is being refreshed or updated.

### Disabled

Grayed out appearance when row interactions are not available.

## Row interactions

### Selection

- Support single or multiple row selection
- Provide clear visual indicators for selected state
- Enable bulk actions on selected rows

### Expandability

- Use clear affordances to indicate expandable content
- Smooth animations for expand/collapse transitions
- Logical organization of expanded content

## Desktop usage and treatment

### Visual hierarchy

- Use consistent row heights for uniform appearance
- Provide adequate spacing between rows for readability
- Maintain alignment across all columns within rows

### Interaction design

- Ensure row interactions are discoverable and accessible
- Use appropriate hover states for interactive elements
- Maintain consistent interaction patterns across all rows

## Mobile usage and treatment

### Responsive behavior

- Adapt row layout for smaller screen sizes
- Consider card-based layouts for complex row data
- Maintain essential information visibility
- Ensure touch targets remain accessible and appropriately sized
