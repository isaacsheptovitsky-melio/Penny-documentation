## Usage

- Display text, numbers, dates, and other data types
- Contain interactive elements like buttons, links, and form inputs
- Support various content layouts within table structure
- Provide consistent data presentation across table rows

## Cell types

### Text cells

Standard cells containing plain text content, such as names, descriptions, or labels.

### Numeric cells

Cells specifically formatted for displaying numbers, currencies, percentages, and other numerical data.

### Action cells

Cells containing interactive elements like buttons, dropdowns, or other controls for row-specific actions.

### Status cells

Cells displaying status indicators, badges, or other state-related information.

### Media cells

Cells containing images, avatars, icons, or other visual elements.

## Content formatting

### Text alignment

- Left-align text content for readability
- Right-align numerical data for easy comparison
- Center-align status indicators and icons

### Content overflow

- Truncate long text with ellipsis when necessary
- Provide tooltips or expandable areas for full content
- Use wrapping for multi-line content where appropriate

## States

### Default

Standard appearance for normal data display.

### Hover

Visual feedback when users interact with the row or cell.

### Selected

Indicates when a row or cell is selected for further action.

### Loading

Shows when cell content is being updated or refreshed.

## Desktop usage and treatment

### Content hierarchy

- Use appropriate typography to establish visual hierarchy
- Maintain consistent spacing within and between cells
- Ensure content is easily scannable across rows

### Interactive elements

- Provide appropriate sizing for buttons and controls
- Use consistent styling for similar actions across rows
- Ensure adequate spacing between interactive elements

## Mobile usage and treatment

### Content adaptation

- Prioritize the most important content for narrow displays
- Consider stacking or hiding less critical information
- Maintain touch-friendly sizing for interactive elements
- Ensure content remains readable on smaller screens
