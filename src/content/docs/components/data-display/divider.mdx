## Usage

**Separating sections**: Use dividers to create a clear boundary between distinct groups of information, such as separating "Vendor details" from "Payment method" in a workflow.

<Aside type="tip">
  Use dividers sparingly. Overusing them can create visual clutter. Generous whitespace is enough to separate related content.
</Aside>

### Vertical

An upright line used to separate inline elements, such as button groups or key data points in a horizontal summary bar.

## Desktop usage and treatment

Use dividers to create logical sections in forms, lists, and content areas. Horizontal dividers work well for separating major content blocks, while vertical dividers help organize inline elements.

## Mobile usage and treatment

On mobile devices, dividers help maintain content hierarchy in condensed layouts. Use them judiciously to avoid creating excessive visual breaks in limited screen space.
