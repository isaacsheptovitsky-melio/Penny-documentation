## Usage

- Use when space is tight but brand recognition is still important
- Use only approved logo assets at the correct size
- Choose variants that maintain visibility on different backgrounds
- Pair with a label only when necessary. Icons should be recognizable on their own

<Aside type="tip">
  Use brand symbols when a full logo isn't needed but clarity and consistency still matter (e.g., sync screens, dropdowns, tiles).
</Aside>

## Use cases

- **Account syncing**: Represent tools like QuickBooks or Xero
- **Payment methods**: Show Visa, Mastercard, etc., in a compact format
- **Dropdowns or tables**: Where space is limited but visual context is useful

## Accessibility

- Ensure color contrast if background is not neutral
- Avoid relying on color alone for interpretation

## Desktop usage and treatment

Brand symbols work well in toolbars, navigation elements, and tables where space efficiency is important. Maintain consistent sizing and spacing to preserve visual hierarchy.

## Mobile usage and treatment

On mobile devices, ensure brand symbols remain recognizable even at smaller sizes. Consider touch target requirements when symbols are interactive elements.
