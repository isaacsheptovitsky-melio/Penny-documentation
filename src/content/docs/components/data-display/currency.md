## Usage

* Use this component to display monetary figures
* Use the `currency` prop to automatically display the correct symbol (e.g., $, €, £) for the specific region.

### Variations

There are 2 variants:

* **Default:** Standard dark text for use on white or light gray backgrounds.
* **Inverse:** White text for use on dark backgrounds, such as a dark sidebar or a specialized branding banner.
