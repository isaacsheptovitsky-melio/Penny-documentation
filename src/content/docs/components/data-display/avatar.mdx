## Usage

- Use an avatar to visually identify a user or company in the interface
- If an image source is not provided or fails to load, the component will display the initials derived from the `name` prop.
- Avatars can include a `badge` to convey status, such as online presence or notifications.
- They are available in multiple sizes to fit various layout contexts, from compact lists to prominent profile headers.

### Sizes

The `size` prop adjusts the dimensions of the avatar for different contexts.

- extra-small: 16px
- small: 24px
- medium (Default): 32px
- large: 56px 
- extra-large: 120px

## Desktop usage and treatment

Avatars are commonly used in headers, user profiles, and lists to provide quick visual identification. Use appropriate sizes based on the importance and context of the user information being displayed.

## Mobile usage and treatment

On mobile devices, ensure avatars maintain adequate touch targets while considering screen space constraints. Smaller sizes may be more appropriate for list items, while larger sizes work well for profile headers.
