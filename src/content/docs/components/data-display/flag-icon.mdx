## Usage

- Use the Flag Icon to visually supplement country information, such as in a currency selector
- A valid two-letter `countryCode` in **ISO 3166-1 alpha-2** format (e.g., 'US', 'GB') is required to display a flag.

## Variants

### Size

The `size` prop controls the dimensions of the icon:

- **small**: Best for dense UIs like table rows or as a prefix in a `TextField`
- **medium** (Default): The standard size for most use cases

## Desktop usage and treatment

Flag icons are useful in forms, dropdowns, and data displays where country context is important. Use the standard medium size for most desktop interfaces and small size for dense data tables.

## Mobile usage and treatment

On mobile devices, ensure flag icons maintain adequate size for touch interactions when used as selectable elements. The circular design helps maintain recognizability at smaller sizes.
