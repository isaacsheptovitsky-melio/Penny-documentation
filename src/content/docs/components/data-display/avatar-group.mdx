## Usage

Use to show a list of users associated with an item.

## Greater than max items

The component displays a maximum of four avatars. Any additional members are summarized in a numerical overflow indicator (e.g., `+3`)

## Desktop usage and treatment

Use avatar groups to represent multiple users associated with a specific item or action. The stacked layout provides an efficient way to display user associations without taking up excessive space.

## Mobile usage and treatment

On mobile devices, avatar groups maintain their compact layout while ensuring touch targets remain accessible. The overflow indicator remains clearly visible and tappable when additional members need to be shown.
