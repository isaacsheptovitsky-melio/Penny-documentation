## Usage

- Use this component to display monetary figures
- Use the `currency` prop to automatically display the correct symbol (e.g., $, €, £) for the specific region.

## Desktop usage and treatment

Currency components should maintain consistent formatting across the application. Use larger sizes for prominent financial displays and smaller sizes for detailed lists and tables.

## Mobile usage and treatment

On mobile devices, ensure currency values remain legible and properly formatted. Consider responsive sizing to maintain readability on smaller screens while preserving the financial context.
