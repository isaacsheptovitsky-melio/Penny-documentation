## Usage

* Use for entering short numeric verification codes (e.g., OTPs)
* Verification code support 4-6 digits
* For passwords use the Secured Text Field component

## Desktop usage and treatment

Verification code fields on desktop should provide clear visual feedback for each digit input and support keyboard navigation between fields. The component should handle paste operations intelligently and provide clear error states when validation fails.

## Mobile usage and treatment

On mobile devices, verification code fields should automatically trigger the numeric keypad and support auto-fill from SMS when possible. The individual input fields should be appropriately sized for touch interactions and provide clear visual feedback for the current input position.
