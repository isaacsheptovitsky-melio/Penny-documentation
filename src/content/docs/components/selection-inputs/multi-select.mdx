## Usage

* Use when users need to select multiple options from a defined set
* Divide content into sections to break it into manageable chunks, making it easier to scan and reducing cognitive load.

> **Note:** For details on sizes, behaviors, and general guidelines applicable to all form components, refer to the [Form Field](https://zeroheight.com/98bb1df01/p/6854b6-form-field) documentation.

## Desktop usage and treatment

Multi Select components on desktop should provide clear visual feedback for selection states and manage the display of multiple selected items effectively. The dropdown should be properly positioned and sized to accommodate the options without overwhelming the interface.

## Mobile usage and treatment

On mobile devices, Multi Select components should adapt to smaller screens while maintaining usability for selecting multiple options. Consider using full-screen overlays or optimized list presentations to improve the selection experience on touch devices.
