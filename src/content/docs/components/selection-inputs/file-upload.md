## Usage

Use the File upload component when:

* When users need to upload one
* When drag-and-drop functionality enhances the experience
* When uploading a file that requires a user-visible preview, use the [Large file uploader](https://zeroheight.com/98bb1df01/v/latest/p/248e54-large-file-upload).
* After a file has been uploaded, the user is allowed to:
    * Delete uploaded file
    * Replace uploaded file
