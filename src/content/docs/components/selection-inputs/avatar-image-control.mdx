## Usage

* Upload an avatar image in order to represent a user
* Use the Circle variant to represent a user
* Use the Square variant to represent a company
* Upon upload, user is able to delete photo

## Desktop usage and treatment

Avatar image controls on desktop should provide clear visual feedback for hover states and ensure the upload/edit functionality is accessible. The edit overlay should appear on hover for better user experience.

## Mobile usage and treatment

On mobile devices, avatar image controls should accommodate touch interactions with appropriate tap target sizes. The edit functionality should be easily accessible without requiring hover states, potentially using long-press or tap gestures to reveal editing options.
