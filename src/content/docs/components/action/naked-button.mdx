## Usage

- Use for non-primary, low-visibility actions
- Supports optional left and right elements

## Left and right elements

The button includes two slots, left and right elements, typically used for icons that either clarify the button's purpose or suggest UI behavior.

### Left element

Used to visually reinforce the buttons meaning, e.g. "+ New vendor".

![Left element](/Penny-documentation/components/naked-button/naked-button-image-10.png)

### Right element

Indicates the button's UI behavior, such as displaying a dropdown.

![Naked button - Right element](/Penny-documentation/components/naked-button/naked-button-image-11.png)
