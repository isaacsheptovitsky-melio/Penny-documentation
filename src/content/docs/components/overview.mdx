Components are reusable UI elements that encapsulate both design and functionality. They serve as the building blocks for creating consistent, accessible, and scalable user interfaces across the Penny ecosystem.

## Component follow our principles

### Consistency

All components follow the same design patterns, interaction behaviors, and visual styling to create a predictable user experience.

### Flexibility

Components support theming, sizing variants, and customization options while maintaining design system integrity.

### Accessibility

Components are built with accessibility as a foundation, including proper ARIA labels, keyboard navigation, and screen reader support.
